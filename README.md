# Python Shopping List Manager
A command-line interface (CLI) tool for managing a grocery shopping list. Stores data in CSV, with features to add items, view the list, update shop names, calculate total spend, and delete items.

## Features
- Add multiple items with name, timestamp, shop, and price.
- Display the list in a formatted table.
- Update shop names across entries.
- Calculate total expenditure.
- Delete specific items.
- Confirmation before exiting.
## Requirements
- Python 3.x (uses built-in `csv` and `datetime` modules).

## How to Run
1. Clone the repo: `git clone https://github.com/yourusername/Python-Shopping-List-Manager.git`
2. Navigate: `cd Python-Shopping-List-Manager`
3. Run: `python shopping_list.py`
4. Follow the menu prompts (1-6).

The app saves data to `list_files.csv` in the same directory.

## Example Usage
- Choose 1 to add items.
- Choose 2 to view.
